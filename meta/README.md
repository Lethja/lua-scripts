This folder contains no unique scripts but instead folders and symbolic links
to scripts in other directories structured in a way that allows them to serve
a purpose.

| Folder Name        | Description                                                                                      |
|--------------------|--------------------------------------------------------------------------------------------------|
| [bench](./bench)   | Contains symbolic links to each script that is needed to run every benchmark in `BENCH.LUA`      |
| [floppy](./floppy) | Contains directories and symbolic links of script structured intended to make floppy disk images |
| [test](./test)     | Unit tests for scripts                                                                           |
